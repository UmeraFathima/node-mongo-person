# Node-Mongo-Person CRUD App

A simple **Node.js + MongoDB** web application that allows you to **Create, Read, Update, and Delete** people.  

The app uses **Express.js** for the server and **Mongoose** for MongoDB interaction. HTML pages are styled with **CSS**.

---

## Features

- List all people (`GET /person`)
- Add a new person (`POST /person`)
- Edit a person (`PUT /person/:id`)
- Delete a person (`DELETE /person/:id`)
- Fields: **Name, Age, Gender, Mobile**
